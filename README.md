# OZETI — Òganizasyon Zetwal Tifi

Official website for **OZETI**, a Haitian non-profit organization dedicated to empowering adolescent girls through education, psychosocial development, and economic independence.

🌐 **Live site:** [zetwaltifi.org](https://zetwaltifi.org)

---

## About

OZETI (Òganizasyon Zetwal Tifi — Organisation Étoile des Filles) was founded on April 3, 2020 in Gonaïves, Haiti by Rose Darline Chatelain. The organization works with girls aged 10–19 across three action areas:

1. **Education & Advocacy** — academic support and rights defense
2. **Psychosocial Skills** — confidence, resilience, and leadership
3. **Economic Empowerment** — vocational training and income-generating activities

Registered with the Haitian Ministry of Social Affairs and Labour (MAST) · N° STZ1-37440

## Tech Stack

- **React 19** + **TypeScript**
- **Vite** (build tool)
- **Tailwind CSS v4**
- **Framer Motion** (animations & page transitions)
- **Lenis** (smooth scrolling)
- **React Router v7**
- **Lucide React** (icons)

## Features

- FR / EN language toggle
- Photo gallery with lightbox
- Smooth page transitions
- Parallax hero image
- Fully responsive
- Deployed on Vercel with custom domain
